Seção 05

Movimento & Interação

Todas as transições usam ease-in-out. A animação principal é whatsapp-animation (fade-in do botão flutuante). Interaja com os exemplos abaixo.

Float Button — whatsapp-animation
@keyframes whatsapp-animation {
  from { opacity: 0%; }
  to   { opacity: 100%; }
}
animation: whatsapp-animation 0.5s ease-in-out
Clique para reanimar o fade-in
Float Button — hover color
transition: background-color 0.3s ease;
/* hover: */
background-color: #170958;
Nav Link — underline slide-in
transition: all 0.3s ease-in-out
/* ::before pseudo-element */
visibility: hidden; opacity: 0;
/* :hover ::before */
visibility: visible; opacity: 1;
background: #fec722; height: 2px;
Institucional Diferenciais
News Card — hover scale + shadow
.news-single-box {
  box-shadow: 0 0 10px rgba(0,0,0,0.13);
  transition: all 0.3s ease-in-out;
}
.news-single-box:hover {
  box-shadow: 0 1px 30px rgba(0,0,0,0.25);
}
.news-img-box img {
  transition: all 0.3s ease-in-out;
}
.news-single-box:hover .news-img-box img {
  transform: scale(1.1);
}
